to build executables for skew and leftist heaps, run:

  make

to build executable for just skew heap, run:

  make skew

to build executable for just leftist heap, run:

  make leftist

--------------------------------------------------------------------------------

to run lab with skew heap:

  ./skew [input_file]

example:

  ./skew data.txt

to run lab with leftist heap:

  ./leftist [input_file]

example:

  ./leftist data.txt
